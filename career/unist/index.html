[UNIST-LG] 고객 발화 Variation 텍스트 생성 (2023.10 – 2024.05) [UNIST-LG] Customer-Utterance Variation Generation (2023.10 – 2024.05)

핵심 역할Key role
기여도 30% — 데이터 수집·생성, 생성 모델 평가 설계30% contribution — data collection/generation, generation-model evaluation design
개요고객 발화 변형(variation) 생성 및 포맷 변경 자동화로 데이터 구축 과정의 인적 자원 투입을 최소화하는 생성형 AI 기반 데이터 증강 파이프라인 개발OverviewA generative-AI data-augmentation pipeline that automates customer-utterance variation and reformatting, minimizing manual effort in dataset construction
1데이터 수집 및 생성Data Collection & Generation
담당 업무Responsibilities
  • ChatGPT를 활용해 few-shot으로 의미는 유지하되 표현이 다른 고객 발화 문장을 반복 생성
  • 생성 문장은 사람 검수를 거쳐 정제 후, 원본 데이터 대비 약 10% 수준으로 데이터 증강 비율 제한
  • 증강 데이터가 학습을 과도하게 주도하지 않도록 원본 텍스트의 분포와 경향성을 유지하면서 표현 다양성만 확장하는 전략 적용
  • Iteratively generated customer utterances with ChatGPT in a few-shot setup — same meaning, varied expression
  • Cleaned generated sentences through human review and capped augmentation at ~10% of the original data
  • Preserved the original text's distribution and tendencies, expanding only expression diversity so augmented data would not dominate training
2생성 모델 평가Generation-Model Evaluation
담당 업무Responsibilities
  • 정성 평가 — ① 표현의 다양성, ② 문장의 완전성, ③ 의미적 일치성 세 기준을 설정하고 사용자에게 각 문장을 1~5점 척도로 평가받음 (랜덤 추출 200개 문장 × 평가자 50명)
  • 정량 평가 — BLEU·ROUGE·BERTScore가 표현 다양성을 충분히 반영하지 못해, ChatGPT를 AI Judge로 활용한 few-shot variation classification 방식 설계
  • Qualitative — defined three criteria (expression diversity, sentence completeness, semantic consistency) and had users rate each sentence on a 1–5 scale (200 random sentences × 50 raters)
  • Quantitative — since BLEU/ROUGE/BERTScore fail to capture variation, designed a few-shot variation-classification scheme using ChatGPT as an AI judge
업무 성과Achievements
  • 정량 평가 정확도 95.77% 달성
  • 사용자 평가에서 평균 3점 이상 비율 81% 달성
  • 해당 과제의 성과로 LG H&A와 추가 산학과제 연장
  • Achieved 95.77% accuracy on the quantitative evaluation
  • 81% of sentences scored an average of 3+ in user evaluation
  • The results led to a follow-on industry-academia project with LG H&A

다국어 멀티모달 융합 — 석사 졸업논문 Towards Efficient Multilingual Multimodal Fusion — M.S. Thesis

핵심 역할Key role
기여도 100% — 개인 연구100% contribution — individual research
개요NC Soft에서의 다국어 처리 경험과 석사 과정의 멀티모달 연구를 결합한 다국어 멀티모달 처리 방법론 연구OverviewA multilingual multimodal methodology combining machine-translation experience from NC Soft with multimodal research during the master's program
1기계번역 기반 다국어 멀티모달 데이터 증강MT-Based Multilingual Data Augmentation
  • 부족한 다국어 데이터를 보강하기 위해 SOTA 기계번역 모델을 활용, 데이터셋을 14개 언어로 확장
  • 번역 결과를 영어로 back-translation한 뒤 원문과의 BLEU score를 기준으로 번역 모델을 선별해 학습 데이터 구성
  • Expanded the dataset into 14 languages with state-of-the-art machine-translation models to compensate for scarce multilingual data
  • Validated quality by back-translating into English and selecting translation models by BLEU score against the source
2다국어 멀티모달 모델링Multilingual Multimodal Modeling
  • 파트 A — 다국어 표현 정렬: 사전 학습된 CLIP 이미지 인코더와 XLM-RoBERTa 텍스트 인코더로 feature를 추출하고, projection layer를 거쳐 다국어 contrastive loss로 image-text pair를 latent space에서 정렬·융합
  • 파트 B — 다국어 캡셔닝 및 생성: 다국어 텍스트와 정렬된 단일 이미지 토큰 벡터를 GPT-2 계열 디코더에 입력, 이미지 기반 캡션 및 Q&A 태스크로 학습
  • Part A — Multilingual Representation Alignment: extracted features with a pretrained CLIP image encoder and an XLM-RoBERTa text encoder, projected each modality, and fused image-text pairs in latent space with a multilingual contrastive loss
  • Part B — Multilingual Captioning & Generation: fed the single image token aligned with multilingual text into a GPT-2-family decoder, trained on image-grounded captioning and Q&A
연구 성과Results
  • 기존 모델 대비 성능 경쟁보다, 단일 image token만으로도 의미 있는 다국어 생성이 가능함을 입증
  • LLM judge 평가에서 다양한 언어의 캡션 평균 3.72/5 달성, 이미지 내용을 반영한 생성 결과 확인
  • Demonstrated that meaningful multilingual generation is possible from a single image token, rather than chasing benchmark gains
  • LLM-judge evaluation scored captions 3.72/5 on average across diverse languages, confirming they reflect image content
  • 방송미디어공학회 2025 게재Published at KSBE 2025

[UNIST-심도컴퍼니] 노이즈 섭동 기반 효과적인 슬로건 생성 [UNIST-Simdo] Effective Slogan Generation with Noise Perturbation

핵심 역할Key role
기여도 100% — 데이터 구축, 모델링, 평가100% contribution — dataset, modeling, evaluation
개요회사의 비전과 브랜드 가치를 담은 슬로건 생성을 자동화하는 새로운 접근법OverviewA new approach to automating slogan generation that captures a company's vision and brand value
1데이터 수집 및 전처리Data Collection & Preprocessing
  • 16개 도메인(food, drinks, tours, cosmetics, technology 등)에 대해 브랜드/회사 설명과 슬로건을 웹 크롤링하여 데이터셋 구축
  • 슬로건 생성의 다양성을 확보하기 위해 하나의 회사에 여러 개의 슬로건이 매칭되도록 구성, 전처리 과정에서 날짜·특수문자 등 불필요한 요소 제거
  • Built a dataset by crawling brand/company descriptions and slogans across 16 domains (food, drinks, tours, cosmetics, technology, …)
  • Matched multiple slogans to each company to secure generation diversity; removed dates, special characters, and other noise in preprocessing
2모델링 및 평가Modeling & Evaluation
  • 트랜스포머 계열 모델 입력 전 input embedding에 가우시안 노이즈를 추가, 문장의 다양성을 조정하는 하이퍼파라미터 도입
  • 표현 다양성과 브랜드 정체성 일치 여부를 함께 평가하는 Cosine Similarity 기반 평가 지표 설계
  • Added Gaussian noise to the input embeddings before the transformer, introducing a hyper-parameter that controls sentence diversity
  • Designed cosine-similarity-based metrics that jointly evaluate expression diversity and brand-identity consistency
연구 성과Results
  • 정량 평가 Description 33.83% → 44.04%, Reference 28.16% → 30.10% 성능 향상으로 기존 방법 대비 SOTA 달성
  • State-of-the-art over prior methods: Description 33.83% → 44.04%, Reference 28.16% → 30.10%
  • CIKM 2023 1저자 게재First-author paper at CIKM 2023

[UNIST-Pyler] CVPR'22 LOVEU Track 3 챌린지 [UNIST-Pyler] CVPR'22 LOVEU Track 3 Challenge

핵심 역할Key role
기여도 60% — 멀티모달 모델링60% contribution — multimodal modeling
개요사용자 시점에서 기계 조작 과정을 단계별로 안내하기 위해, 설명 영상과 스크립트를 학습하여 사용자 질문에 응답하는 방법론을 제안한 CVPR 2022 LOVEU Workshop 챌린지OverviewThe CVPR 2022 LOVEU workshop challenge: answer user questions step-by-step by learning from instructional videos and scripts, guiding machine operation from the user's viewpoint
1멀티모달 모델링Multimodal Modeling
  • 영상, 텍스트 스크립트, 사용자 질문, 사용자 이미지, 정답 후보 등 다양한 입력을 각각 ViT 또는 BERT로 feature 추출, modality별 정보를 독립적으로 인코딩
  • 시각-텍스트 간 의미적 연결을 강화하기 위해 Cross Multi-Head Attention 구조를 적용해 modality 간 상호작용 학습
  • 모든 feature에 GRU를 적용해 시간 정보를 반영, 사용자의 질문에 대해 단계적인 응답이 가능하도록 설계
  • 최종적으로 MLP를 통해 정답 후보 중 하나를 선택
  • Encoded each input — video, text script, user question, user-view image, and answer candidates — independently with ViT or BERT
  • Applied cross multi-head attention to strengthen the semantic connection between visual and textual modalities
  • Applied a GRU across all features to inject temporal information, enabling step-wise responses to user questions
  • Selected the final answer among candidates with an MLP head
연구 성과Results
  • 베이스라인 대비 평균 Recall 0.075 향상
  • CVPR 2022 LOVEU Workshop 챌린지 2등 입상
  • 약 100개의 소량 학습 데이터에서도 효과적인 학습 구조 제시
  • Improved mean recall by 0.075 over the baseline
  • 2nd place at the CVPR 2022 LOVEU workshop challenge
  • Demonstrated an architecture that learns effectively from only ~100 training samples
  • CVPRW 2022 기술 보고서CVPRW 2022 technical report
